Express.Js Command line and Contents

Initial npm to the project

  • npm init -y

    • Initial npm Express

  • npm i express

    • check Server Running: on Browser:

  • localhost:3000

    • Stop Server Running:

  • Ctrl + C

    • Check which ports are listening on our computer

    Windows:

  • netstat -ano | findstr "LISTENING"

    • Mac/Linux:

  • sudo lsof -i -p -n | grep LISTEN

    • HyperText Transfer Protocol

  • HTTP

    • Request Vocab:

  • GET //Request Resource
  • POST //Sending Resource
  • PUT //Replace Resource
  • PATCH //Patch up a Resource (one part)
  • DELETE //Delete Resource

    • Automatically restarts Node.js apps when files change during development

  • nodemon File-Name // index.js
  • npm i -g nodemon // Global Running
  • nodemon index.js
